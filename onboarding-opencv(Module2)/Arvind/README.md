# Module 2 – OpenCV Image Processing

## Overview

This module focuses on fundamental image processing and computer vision techniques using OpenCV. The assignment consists of ten practical tasks implemented using personally captured images and videos.

The implementation is contained inside the `Arvind` folder.

## Contents

```
Module-2/
│
└── Arvind/
    ├── answer.ipynb
    ├── raw_captures/
    ├── output-task5/
    ├── OpenCV Free OpenCV Bootcamp Certificate.pdf
    └── README.md
```

## Tasks Completed

- Task 0 – Image Capture
- Task 1 – Color Space Conversion
- Task 2 – Image Blending using ORB Feature Matching
- Task 3 – Geometric Transformations
- Task 4 – Noise Addition and Image Filtering
- Task 5 – Edge Detection and Contour Detection
- Task 6 – Thresholding and Morphological Operations
- Task 7 – ORB Feature Matching Across Images
- Task 8 – Face Detection using Haar Cascade
- Task 9 – Real-Time Video Processing
- Task 10 – Camera Calibration and Lens Undistortion

## Files

- `answer.ipynb` – Complete notebook containing Tasks 0–10
- `raw_captures/` – Original images and videos captured for the assignment
- `output-task5/` – Output video generated and Screen Recording of Task 5
- `OpenCV Free OpenCV Bootcamp Certificate.pdf` – OpenCV bootcamp completion certificate
